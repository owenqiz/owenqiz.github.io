A Simple Linear Regression

February 7, 2019 - 1 minutes
Statistics R Markdown

We build a linear regression below.

Simple linear regression is of the form \[Y_i = \beta_0 + \beta_1X_i + \varepsilon_i\] where \(\varepsilon_i \sim N(0,\sigma^2)\), with \(\sigma\) be a constant.

fit = lm(dist ~ speed, data = cars)
b = coef(summary(fit))
plot(fit)

The slope of the regression is 3.9324088.
